# homebrew-a2fuse

Homebrew tap for [a2fuse](https://github.com/markassad/a2fuse).

## Installation

```sh
brew tap markassad/a2fuse
brew install a2fuse
```

## Usage

```sh
mkdir -p ~/mnt/apple2
a2fuse mount image.po ~/mnt/apple2
```

See [a2fuse documentation](https://github.com/markassad/a2fuse#readme) for more details.
